# React Frameworks Hands-On Assignment

## Overview
This repository contains hands-on projects for learning different React frameworks. The goal of this assignment is to explore **Next.js**, **React Router (Remix replacement)**, **Gatsby**, and **Expo**, understanding their features and development workflows.

## Folder Structure

React-Frameworks-Assignment/
├─ your-next-app/ → Next.js project
├─ your-remix-app/ → React Router project (Remix replacement)
├─ your-gatsby-app/ → Gatsby project
└─ your-expo-app/ → Expo project

---

## 1. Next.js

### Run locally:

```bash
cd your-next-app
npm install
npm run dev
```
Open your browser at: http://localhost:3000

---

## 2. React Router (Remix replacement)

### Run locally:

```bash
cd your-remix-app
npm install
npm start
```
Open your browser at: http://localhost:3000

---

## 3. Gatsby

### Run locally:

```bash
cd your-gatsby-app
npm install
npm run develop
```
Open your browser at: http://localhost:8000

---

## 4. Expo

### Run locally:

```bash
cd your-expo-app
npm install
npx expo start
```
Use Expo Go on your mobile device or an emulator to preview the app.
Follow the QR code in the terminal or browser to open the project.
